# 明日藏寶圖

明日藏寶圖是一段面向香港使用者的生前遺產整理旅程。它以可滾動的成熟尋寶地圖，引導使用者看見自己累積的財產、債務、想照顧的人，以及是否希望作出公益遺贈。

## 产品边界

- 這不是遺囑、可簽署法律文件或個人法律意見。
- 正式安排須由香港執業律師根據使用者的完整情況確認。
- 產品只處理財產、債務、親友分配方向與公益遺贈，不處理醫療、殯葬或授權安排。
- 沒有帳戶、後端、個人結果分享連結或雲端儲存。

## 私隐设计

填写内容只保留在页面运行内存中。轻量草稿会去除精确金额；完整草稿以使用者密码在装置内用 AES-GCM 加密。密码不会保存，遗忘后无法恢复完整草稿。个人 PDF 默认隐藏金额，给律师看的 PDF 则保留使用者选择填写的细节。

## 法律来源

网页中的香港法律路标必须引用支持该段窄范围说明的官方来源，并记录最后核对日期。路标只帮助使用者发现需要询问律师的问题，不作个人法律结论。

## 本机运行

需要现代浏览器及 Node.js 运行测试：

```powershell
python -m http.server 8000
npm.cmd test
npm.cmd run check
```

打开 `http://127.0.0.1:8000`。这个项目是静态网站，可部署到 GitHub Pages。
