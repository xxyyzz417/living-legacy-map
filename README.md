# 明日藏寶圖

明日藏寶圖是一段面向香港使用者的生前遺產整理旅程。它以可滾動的成熟尋寶地圖，引導使用者看見自己累積的財產、債務、想照顧的人，以及是否希望作出公益遺贈。

## 產品邊界

- 這不是遺囑、可簽署法律文件或個人法律意見。
- 正式安排須由香港執業律師根據使用者的完整情況確認。
- 產品只處理財產、債務、親友分配方向與公益遺贈，不處理醫療、殯葬或授權安排。
- 沒有帳戶、後端、個人結果分享連結或雲端儲存。

## 私隱設計

填寫內容只保留在頁面運行記憶體中。輕量草稿會去除精確金額；完整草稿以使用者密碼在裝置內用 AES-GCM 加密。密碼不會保存，遺忘後無法恢復完整草稿。個人 PDF 預設隱藏金額，給律師看的 PDF 則保留使用者選擇填寫的細節。

## 法律来源

網頁中的香港法律路標必須引用支持該段窄範圍說明的官方來源，並記錄最後核對日期。路標只幫助使用者發現需要詢問律師的問題，不作個人法律結論。

## 本机运行

需要现代浏览器及 Node.js 运行测试：

```powershell
python -m http.server 8000
npm.cmd test
npm.cmd run check
```

打开 `http://127.0.0.1:8000`。这个项目是静态网站，可部署到 GitHub Pages。
